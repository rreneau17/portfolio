Featured Project



www.groupstrengthsf.com

Group Strength Social Fitness is a custom application that allows one the convenience of entering strength training logs directly into a Fitbit watch.  Logs are securely submitted to a responsive and RESTful web application where one can track progress, create an exercise routine library, and potentially share results with others.  The web app was developed using Node.js, Express, PostgreSQL, Sequelize, Handlebars, JavaScript, jQuery, Bootstrap, HTML, and CSS.
